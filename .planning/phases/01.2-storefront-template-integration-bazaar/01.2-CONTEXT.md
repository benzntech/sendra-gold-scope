# Context: Phase 1.2 (Storefront Template Integration - Bazaar)

## Source
Decision to replace the existing storefront codebase with the premium Bazaar Next.js template.

## Key Objectives
1. **Scaffold & Configure**: Setup the Bazaar template in `sendra-storefront/`.
2. **Branding & Theme**: Integrate Sendra Gold brand identity (colors: Sendra Red `#C92127`, Gold `#C5A059`, etc.) into the Bazaar UI components.
3. **API Integration Layout**: Establish the layout shell for the classic **Market 1** shopping storefront.

## Decisions
- Replace storefront contents with the Bazaar Next.js template.
- Retain existing Bagisto Laravel API endpoints as the backend source.
- Brand colors remain Sendra Red (#C92127) for CTAs, Gold (#C5A059) for accents.
